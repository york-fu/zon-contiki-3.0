/****************************************工程说明*********************************************
*sensor-a协议：
*
*    属性	  参数	 权限	说明
*    温度值	  A0	 R	存储温度值，浮点型：0.1精度
*    湿度值	  A1	 R	存储湿度值，浮点型：0.1精度
*    光照值	  A2	 R	存储光照值，浮点型：0.1精度
*    空气质量值	  A3	 R	存储空气质量值，整型
*    气压值	  A4	 R	存储气压值，浮点型：0.1精度
*    三轴值	  A5	 R	存储跌倒状态，0：未跌倒，1：跌倒
*    距离值	  A6	 R	存储距离值，浮点型：0.1精度
*
*    继电器	  D1	 RW	Bit6、Bit7控制继电器状态，用OD1/CD1改变状态
*    上报使能	  D0
*             (OD0/CD0)	 RW	上报状态，bit0~bit3分别对应A0~A1；0表示不允许上报，1表示允许上报，通过OD0/CD0进行状态控制
*    操作编码	OD1/CD1	 RW	对D1的位进行操作，CD1表示位清零操作，OD1表示位置一操作
*    上报间隔	  V0	 RW	设置为上报时间间隔，单位秒
*    设备类型	  TYPE	 R	12601
*
*************************************语音识别（A7）*******************************************
*    1------------打开台灯                  2------------关闭台灯
*    3------------打开窗帘                  4------------关闭窗帘
*    5------------打开风扇                  6------------关闭风扇
*    7------------打开空调                  8------------关闭空调
*    9------------打开加湿器               10------------关闭加湿器
*   11------------自动模式                 12------------手动模式
*
*********************************************************************************************/
