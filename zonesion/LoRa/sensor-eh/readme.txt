/****************************************工程说明*********************************************
* SENSOR-EH协议：
*
*    属性	  	参数	 	权限	说明
*    卡号		A0		-	字符串（主动上报，不可查询）
*    卡余额		A2		R	整型，范围0~8000.00，手动查询
*    ETC杆开关		D1(OD1/CD1)	RW	D1的Bit0表示ETC杆开关0（关闭），1（抬起一次3s自动关闭，同时bit0置0）
*    充值金额		V1		RW	返回充值状态，0/1，1表示操作成功
*    扣款金额		V2		RW	返回扣款状态，0/1，1表示操作成功

*    设备类型	  TYPE	 R	12606
