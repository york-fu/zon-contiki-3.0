/****************************************工程说明*********************************************
*sensor-b协议：
*
*    控制设备	  D1	 RW	Bit0、Bit1控制RGB灯状态，用OD1/CD1改变状态
                                Bit2控制步进电机正转/反转状态，用OD1/CD1改变状态       
*                               Bit3控制风扇/蜂鸣器状态，用OD1/CD1改变状态
*                               Bit4、Bit5控制LED1/LED2状态，用OD1/CD1改变状态
*                               Bit6、Bit7控制继电器状态，用OD1/CD1改变状态
*    上报使能	  D0
*             (OD0/CD0)	 RW	上报状态，bit0~bit3分别对应A0~A1；0表示不允许上报，1表示允许上报，通过OD0/CD0进行状态控制
*    操作编码	OD1/CD1	 RW	对D1的位进行操作，CD1表示位清零操作，OD1表示位置一操作
*    上报间隔	  V0	 RW	设置为上报时间间隔，单位秒
*    设备类型	  TYPE	 R	12602
*
*********************************************************************************************/
